## 最近完成的事（2026-05-29 ~ 2026-06-05）

### 推論管線修正
- **GATv2 CSR Bug 修正（PR 1）**：推論時 `load_kg_edges()` 用 Python loop，batch=128 的邊索引超過 127 全被丟棄，GATv2 等於沒有運作。改成 CSR 快速版後 GATv2 才真正發揮作用
- **每日重建 rolling correlation 邊**：Step 1 完成後、Step 3 前插入 `update_correlation_edges()`
- **model_tracker.jsonl（PR 2）**：每日推論後 append 一行 JSON（date / val_ic / top50 / ic_analysis / scanner_summary / duration）+ `load_tracker_history()` 讀取函式

### V6.2 訓練（第一輪）
- 加入 RS_5d / RS_20d / RS_60d（INPUT_DIM 56→59）
- Best IC = **0.0950**（epoch 13），比 V6.1 的 0.0825 明顯提升
- Scale gate 觀察：Short: 0.010 / Mid: 0.001 / **Long: 0.989** → 極端偏 Long

### V6.2 訓練（第二輪，Padding Mask 啟用）
- `USE_PADDING_MASK = True`，Long branch 的 padding 位置輸出乘 0 截斷梯度
- Short / Mid 不需要 mask（它們取最後 20/60 步，全是真實資料）
- **目標：讓 scale_gate 三個分支均衡（各 0.25~0.4）**，讓 RS 短期特徵真正被學到

### 訊號系統 V6.2 整修
- `signal_conditions.py` — 150 分進場評分 + 四層退場 + Trailing Stop + 進場理由記憶
- `pattern_scanner.py` — 5 多方 + 2 空方型態，含 `failure_stop`
- `sim_engine_v3.py` — 有狀態日更機器人，`sim_state.json` 持久化
