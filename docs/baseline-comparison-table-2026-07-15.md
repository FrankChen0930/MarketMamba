## 判讀：方向二要回答的問題，答案是什麼

**Q：用可解釋性換到多少效益？——答案是負的。** 在這個 harness 上（5d、台股全市場、rank label），可解釋性最高的 Ridge 已勝 Mamba +0.015；沿光譜往黑箱走（GBDT、GRU），IC 增量合計只有 +0.010。**訊號層排序**：GRU > GBDT > Ridge >> Mamba；**落袋排序**：GRU > Ridge > GBDT（Mamba 未跑組合層）。

三條互相獨立的證據收斂到同一結論：
1. **模型形式只在 ±0.01 內移動**（Ridge +0.1015 → GBDT +0.1098 → GRU +0.1113），而「當日快照 → 加工程化時序特徵」帶來 +0.015（0.0861→0.1015）、「z-score 目標 → rank 目標」帶來近翻倍（Mamba 端 0.049→0.0951）——**報酬率最高的投入是特徵與 label 設計，不是架構**。
2. **序列 vs 扁平幾乎無差**（GRU vs GBDT 僅 +0.0015）：好的 lag/rolling 特徵工程 ≈ 序列模型自己學時序。
3. **Mamba 架構紅利不成立**（階 3 的隔離問題）：~49K 參數 GRU 用同特徵、同 window 60、同 label 勝 1.66M 參數 v6_short（+Mamba SSM、+GATv2、+listnet）+0.024。「任何序列模型皆可」成立。

**教訓（GBDT 階換來的，做任何對照表都要保留）：IC 排名 ≠ 落袋排名。** GBDT IC 比 Ridge 高卻在組合層反輸 7.9pp（增量集中小型股與排名中段、頭部換手更高）。本表因此把訊號層與組合層並排，單看 IC 一欄會下錯結論。

**這不等於 Mamba/DL 線沒價值**：尚未對照的包括 60d horizon、Uncertainty 校準（方向三-C 顯示 20d SQ 有實證增量）、KG 在事件傳導上的作用。但 **5d 短線這條線上，「簡單模型 + 好特徵」是效率最高的路徑**，與現代橫斷面選股文獻共識一致。

對 Mamba 偏鬆（epoch 峰值選擇）；train/test 無 purge；test 窗與 Phase 3 harness 重用（B9 慢性）；存活者偏差未量化；GRU 有 4 組網格的輕微多重測試優勢；各階網格皆小（未調到極限，但 val IC 平坦顯示邊際有限）。逐條展開見各階報告。
