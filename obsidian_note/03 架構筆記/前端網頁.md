## 兩套前端

MarketMamba 有兩個獨立的前端，功能有高度重疊，但用途不同：

| | Web Dashboard | PersonalOS |
|--|--------------|------------|
| 技術 | Vite + React | Electron + React |
| 部署 | Vercel（公開） | 本機桌面應用 |
| 資料來源 | Render 後端 API | 直接讀本機 + Render API |
| 持倉同步 | Shioaji API（永豐）| Shioaji（永豐）+ esun_trade（玉山） |
| 後台管理 | ❌ 無 | ✅ AdminOps 頁面 |
| 目標用戶 | 對外展示、未來可分享 | 自己日常使用 |

4. **系統介紹**：推論流程 6 步圖、模型架構 monospace block、技術規格表（8 行）、技術棧 badge

### 量化分析（`/quant`）— 5 個子分頁

1. **技術指標**：大盤 MA/RSI/MACD/KD/Bollinger/ATR 表 + 雷達圖
2. **籌碼面**：三大法人淨買超 + 外資走勢 + 產業資金輪動
3. **市場廣度**：漲跌家數柱狀圖 + A/D Ratio 趨勢
4. **模型 Alpha**：產業平均 Alpha 橫向圖 + 信心分佈 + Alpha 統計
5. **傳統型態學**：5 多方 + 2 空方型態，雙確認（型態分數 ≥60 且 Alpha ≤200）

### 持倉追蹤（`/portfolio`）

### AdminOps（`/market/admin`）— 後台管理

這是讓我能在不打開終端機的情況下直接控制推論的介面。

**兩個主要功能區**：

**① 純模型推論**（單獨補跑）
- 「立即執行推論」按鈕 → 呼叫 `window.electronAPI.runMmInference()`
- 可勾選「跳過 git push」（測試用）
- 即時進度條（解析 `[N/10]` log 格式）
- 即時輸出終端（scrollback 2000 行）
- 底部歷史 log（`V6/logs/inference.log` 末 300 行）

**② 完整每日自動化**（補跑整天）
- 等同執行 `run_daily.bat`
- 依當下時間自動判斷 morning / full / news-only 模式
- 包含：MarketMamba V6 推論 + 永豐持倉同步 + 新聞分析
- 同樣有即時進度條 + 輸出終端

**鎖檔偵測**：
- 啟動時檢查是否有遺留的 lock 檔（PersonalOS 被關閉但 WSL 程序還在跑）
- 顯示警告 + 建議終端機指令確認

### Portfolio（PersonalOS 版）— 持倉追蹤

PersonalOS 的持倉頁比 Web 版功能更完整：

**多帳戶支援**：
- 永豐證券（Shioaji 直連）
- 玉山證券（`esun_trade` SDK，可手動觸發「⚡ 立即同步」）
- 合計 tab（兩帳戶合併）

**退場條件 Modal**（點擊持倉列開啟）：
- 持有成本 / 現價 / 止損參考 / 報酬率 快照
- 4 個退場條件逐一顯示觸發狀態
- 最終建議：🔴 立即退場 / 🟡 密切觀察 / 🟢 繼續持有

**風險分數（0–100）**：
- `exitSignal` 觸發 +40
- 報酬率 < -8% +25；< -4% +12
- model_signal = SELL +20；HOLD +8

**交叉比對**：從 Render API 拉 Scanner exit_signals，有觸發的持倉顯示 🔴 退場標籤

## 後端（`app/backend/`）

**框架**：FastAPI + Uvicorn，部署在 Render

### 資料流

```
GitHub (V6/results/) ← git push（每日推論後）
    ↓ 每小時 TTL 快取（asyncio.Lock 防競態）
  Render 後端
    ↓ REST API
  前端（Vercel Web / PersonalOS Electron）
```

**強制刷新**：`POST /api/signals/cache/refresh`

### 主要路由

| 路由 | 說明 |
|------|------|
| `GET /api/signals/daily` | `df_kelly.csv` + `action_signals.json` |
| `GET /api/signals/scanner` | Scanner 訊號（BUY / EXIT / WATCH） |
| `GET /api/market/overview` | 大盤即時行情（Yahoo v8 直連） |
| `GET /api/quant/analysis` | 技術面 + 籌碼面分析 |
